'From' Email address

The plugin sends an email to the end-users and the responders when a new conversation is
started.

Enter a valid 'From' email address in the format 'name <email-address>' for this email.
For example, 'noreply <noreply@your-domain.com>'.

Note that most hosting services will require this to be a valid email address to ensure delivery.

Folder for screen templates for the front end

Use the default supplied html template files or provide your own folder name here to change the design of the front end screens.

Supplied html templates files are located in:
  templates/bootstrap/ (if you are using bootstrap) and
  templates/simple-html5/ (if you are not using bootstrap).

The default setting is templates/simple-html5/
